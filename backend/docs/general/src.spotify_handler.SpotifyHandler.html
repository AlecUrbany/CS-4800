View In Hierarchy

A class to handle all Spotify API interactions.

There are two types of Spotify clients; a base client and a user client. A base client uses the default SentiSounds Spotify account to perform Spotify operations. The user client will be able to return results and recommendations directly via the Spotify API.

The from_base_client or from_user_client classmethods are the easiest ways to get a SpotifyHandler instance
