class YoutubeHandler:

View In Hierarchy

A handler for interacting with the YouTube API.

Contains a definition for a static _client_instance. This should only be accessed via the get_client() function, which will automatically fill this field if it does not yet exist. Any other accesses to this instance are unsafe and should not be used.
